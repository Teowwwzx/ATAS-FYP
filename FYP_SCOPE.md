Scope Summary

- Build a managed marketplace web app for Malaysia’s academia–industry ecosystem using Python backend and React frontend.
- Provide standardized expert profiles, streamlined booking workflow, and in‑platform communication.
- Implement an AI‑enhanced search to accurately match experts with student initiatives.
- Include flexible fintech module for applicable fees and admin tools (users, organizations, events, categories, notifications, audit logs).
- Deliver an MVP only; exclude monetization strategy, advanced social networking, global rollout, long‑term ops/marketing, logistics, or large‑scale impact measurement.
